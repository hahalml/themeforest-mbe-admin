A) HTML Structure - top

This theme is a fixed/liquid layout with 16 columns. Everything is stored in a div with the class "page". To make the div liquid you add the class liquid to that div. By default it is fixed. The general template structure is the same throughout the template. Here is the general structure.

Instead of $THEME you must write the theme class name. Available options are:

  • soft-green
  • crimson-orange
  • soft-blue
  • soft-red
  • soft-purple
  • soft-yellow
  • blue-gray

By default the theme is soft-green.

B) CSS Files and Structure - top

In this template the CSS files can be split into 4 types: google fonts, plugin requirements, main css and theme css. All need to be included all the time (they are mandatory).

Instead of $THEME you must write the theme class name. Available options are:

  • soft-green
  • crimson-orange
  • soft-blue
  • soft-red
  • soft-purple
  • soft-yellow
  • blue-gray

If you want to use a theme you need to include its own CSS file (except for soft-green which is default).

C) JavaScript - top

The Javascript files can be split into 3 types: jquery, plugins and main js. We recommend you incude all of them, but if you feel that you want to use a different plugin or something you can replace at your will.

The Javascript libraries & plugins used:

  1. jQuery
  2. jQuery UI: Sortable, Accordion, Autocomplete, Progressbar, Shake Effect
  3. jQuery jdPicker
  4. jQuery Uniform
  5. jQuery HotKeys
  6. jQuery Visualize
  7. Fancybox
  8. jWysiwyg
  9. rotate3Di
  10. jQuery DataTable
  11. jQuery Validation
  12. jQuery Detect Mobile Browser
  13. FullCalendar
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  2. jQuery UI provides abstractions for low-level interaction and animation, advanced effects and high-level, themeable widgets, built on top of the jQuery JavaScript Library, that you can use to build highly interactive web applications.
    • Sortable: The jQuery UI Sortable plugin makes selected elements sortable by dragging with the mouse.
    • Accordion: Make the selected elements Accordion widgets.
    • Autocomplete: Autocomplete, when added to an input field, enables users to quickly find and select from a pre-populated list of values as they type, leveraging searching and filtering.
    • Progressbar: The progress bar is designed to simply display the current % complete for a process.
    • Shake Effect: The effect is used on the login page when the user enters the wrong credentials.
  3. jdPicker is a light datepicker (~20Ko) that allows you to easily customize its behaviour and style.
  4. jQuery Unfiorm is a jQuery Plugin that helps you style checkboxes, dropdown menus, radio buttons and file upload inputs.
  5. jQuery HotKeys is a jQuery plugin for hooking keyboard events.
  6. jQuery Visualize is a jQuery plugin that groks HTML tables and generates lovely charts from it.
  7. FancyBox is a tool for displaying images, html content and multi-media in a Mac-style "lightbox" that floats overtop of web page.
  8. jWysiwyg is minimalistic WYSIWYG editor built as jQuery plugin.
  9. Rotate3Di is a jQuery Effect Plugin that makes it possible to do an isometric 3D flip or 3D rotation of any HTML content.
  10. DataTables is a plug-in for the jQuery Javascript library. It is a highly flexible tool, based upon the foundations of progressive enhancement, which will add advanced interaction controls to any HTML table.
  11. This jQuery plugin makes simple clientside form validation trivial, while offering lots of option for customization.
  12. jQuery.browser.mobile will be true if the browser is a mobile device.
  13. FullCalendar is a jQuery plugin that provides a full-sized, drag & drop calendar like the one below.

The main.js is a well structured object that you can use to expand or customize the admin. For example: To initialize the form elements you could use: 

mbe.init.form.all();

To initialize just the date picker you could use:

mbe.init.form.date();

To initialize the grap:

mbe.init.graph.ready();

The main Javascript file is structured in such a way that it is easy to use and easy to be expanded.

F) Additional Unique Features - top

The search on the top right is one of the greatest feature of this admin. Working in MacOS, we have, almost always, used spotlight to open apps. This is a copy of spotlight. It automatically takes the pages of the administration from the menu and using autocomplete you can find any page you want really fast. It also has a shortcut. In all browsers it's CTRL + Space except for Firefox on Mac which has ALT + SPACE.
